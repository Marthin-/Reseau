
-Installer(dossier Installation) : 
	-Installer SDL Mixer 1.2
		=> sudo apt-get install libsdl-mixer1.2 libsdl-mixer1.2-dev
	-Installer SDL Image 1.2
		=> sudo apt-get install libsdl-image1.2 libsdl-image1.2-dev
	-Installer SDL TTF 2.0
		=> sudo apt-get install libsdl-ttf2.0-dev libsdl-ttf2.0-0
	-Installer SDL 1.2
		=> sudo apt-get install libsdl1.2debian libsdl1.2-dbg libsdl1.2-dev
	-Commande rapide 
		=>sudo apt-get install libsdl-mixer1.2 libsdl-mixer1.2-dev libsdl-image1.2 libsdl-image1.2-dev libsdl-ttf2.0-dev libsdl-ttf2.0-0 libsdl1.2debian libsdl1.2-dbg libsdl1.2-dev
-Compiler :
	-make
-Effacer :
	-make clean
-Lancer :
	-./exec
-Utilisation :
	-Direction
		-SI VOUS N'AVEZ PAS DE PAVE NUMERIQUE
			-gauche : fleche de gauche
			-droite : fleche de droite
			-haut : fleche du haut
			-bas : fleche du bas
		-SINON
			-gauche : 4
			-droite : 6
			-bas : 2
			-haut : 8
			-haut droite : 9
			-haut gauche : 7
			-bas droite : 3
			-bas gauche : 1
	-Canne a peche : Positionner votre souris sur la cible puis Touche C
	-Filet : Positionner votre souris sur la cible puis Touche F
	-Afficher sac : Touche S
	-Afficher information/animaux : Left CTRL + Touche i
	-Construire pont : Left CTRL + Touche Direction
	-Redémarrer partie : Touche R
	-Passer le tour : Touche Entrée
	-Quitter : Touche Echap
-PS : 
	-Lors du lancement un écran d'accueil se lance, n'attendez pas la fin de cette écran,
	vous devez répondre aux questions posées par la console
